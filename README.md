# Aether Billing monorepo

This repository powers **billing.aetherframeworks.dev** and **terminal.aetherframeworks.dev**.

| Directory | Purpose |
|-----------|---------|
| **aether-billing-core** | Node.js billing API (Stripe Connect, installments, terminal routes) |
| **aether-frameworks-billing** | WordPress/WooCommerce plugin for Aether checkout |
| **aether-terminal-web** | Node + Vite PWA for merchants on iPhone — served at **terminal.aetherframeworks.dev** |

## Terminal web

See [aether-terminal-web/README.md](aether-terminal-web/README.md) and [aether-terminal-web/HOSTINGER-NODE-DEPLOY.md](aether-terminal-web/HOSTINGER-NODE-DEPLOY.md) for local dev and Hostinger Node deployment.
